Use Python 3.10.11
Use Django 4.2.6

create folder
Open terminal in that directory
Download project into said folder
make virtual environment by typing "python -m venv env"
In the terminal run these commands;
	"pip install django"
"pip install django-crispy-forms==1.9.2"
"pip install Pillow"
Then type “env\Scripts\activate”
“ cd .\learning_system\”
“python manage.py makemigrations”
“python manage.py migrate”
“python manage.py runserver”

This will then prompt a link “ http://127.0.0.1:8000/” which you should click and will direct you to the site
